Создание телеграм бота с рандомными фильмами.

Не знаете, что посмотреть сегодня вечером? Запускайте моего бота и получите случайный фильм, который скрасит вам вечер!)

Как запустить:
В терминале (Я использовал Vscode с установленным python) необходимо подключить библеотеку telebot.

Команда:  pip install pyTelegramBotAPI

Далее запустить наш код, и зайти в телеграм. И в поиске найти: @RndmMovieBot

(Вообще еще необходимо вставить apitoken в код, ткк он был убран из соображений безопасности, ибо в гитхабе все публичное и лучше в лишний раз не сливать свои данные))
После чего ботом можно пользоваться)
